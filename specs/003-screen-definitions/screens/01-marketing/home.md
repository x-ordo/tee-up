# Screen: Landing - Golfers (고객 랜딩)

**Route**: `/`
**Status**: Implemented
**Last Updated**: 2025-12-30
**Priority**: P1
**Dependencies**: Chat, AI Recommendation

---

## 1. 화면 개요

### 목적
고객(골퍼)이 불안 없이 신뢰할 수 있는 프로를 빠르게 찾고 상담을 시작하도록 유도한다.

### 사용자 흐름
```
직접 방문/검색 → / → 신뢰 카피 확인 → 채팅 시작 → AI 추천 결과 확인
```

### 접근 조건
- **인증 필요**: No
- **권한 수준**: guest

---

## 2. 레이아웃 구조

### 섹션 구성 (불필요 요소 제거)

1. **Trust Hero**
   - 헤드라인: "검증된 프로에게, 가장 빠른 매칭"
   - 서브카피: "AI 추천 + 운영팀 검수로 신뢰도를 높였습니다. 불필요한 절차 없이 상담만 시작하세요."
   - CTA: "프로 추천 받기" (Primary 단일 CTA)
2. **Trust Proof**
   - 타이틀: "검증된 프로만 추천합니다"
   - 카피: "프로 인증 서류 확인 · 활동 지역/경력 검증 · 운영팀 1차 리뷰를 통과한 프로만 노출됩니다."
3. **Responder Clarity**
   - 타이틀: "누가 답변하나요?"
   - 카피: "초기 응답은 운영팀(컨시어지)이 진행합니다. 매칭 확정 후에는 해당 프로가 직접 소통합니다."
4. **Chat Entry**
   - 임베디드 채팅/상담 진입 영역
   - 첫 문의 문장 템플릿 제공
5. **AI Recommendation Strip**
   - 온톨로지 기반 추천 프로 3-6명
   - 핵심 태그/전문 분야/경력 하이라이트
6. **FAQ (Minimal)**
   - 상담 비용/추천 기준/취소 정책/응답 SLA 요약
   - FAQ 카피
     - Q: "상담 비용이 있나요?" → A: "첫 상담은 무료입니다. 세부 비용은 프로별로 안내됩니다."
     - Q: "추천 기준은 무엇인가요?" → A: "실력/경력, 활동 지역, 선호 스타일을 종합해 매칭합니다."
     - Q: "원하지 않으면 취소할 수 있나요?" → A: "상담 단계에서는 언제든 중단 가능합니다."
     - Q: "응답은 얼마나 걸리나요?" → A: "평균 24시간 이내 1차 응답을 목표로 합니다."
7. **Compact Footer**
   - 고객 보호 약속, 개인정보 보호 안내

## 4. 데이터 요구사항

- 추천 프로 목록 (AI/온톨로지 매칭)
- 검증 상태/경력 요약
- 채팅 시작용 프로필/세션 ID
- 응답 주체 안내 문구 (운영팀/프로 역할 구분)
- FAQ 데이터 (가격/과정/취소/환불, SLA)

---

## 5. 인터랙션 정의

### 5.1 상담 시작
1. **Given** 고객이 랜딩에서 신뢰 정보를 확인하고
2. **When** "프로 추천 받기" CTA 또는 채팅 진입을 클릭하면
3. **Then** 상담 세션 생성 또는 채팅창 열림

### 5.2 AI 추천
- 채팅 입력/선택값을 기반으로 추천 카드 업데이트
