# TEE:UP 디자인 원칙 v1.0

## 1. Design North Star

### Calm Luxury (차분한 고급스러움)
- **정의**: 불필요한 장식을 배제하고, 여백과 정제된 타이포그래피로 본질적인 가치에 집중합니다. 과시적인 화려함 대신, 은은하고 세련된 분위기를 통해 '프리미엄' 경험을 제공합니다.
- **타겟 페르소나**: VIP 골퍼, 프로 골퍼

### Magazine-First (매거진 우선)
- **정의**: 모든 프로필과 콘텐츠는 고품질의 골프 매거진 페이지처럼 구성됩니다. 시각적 요소가 정보를 이끌며, 풍부한 이미지와 영상이 텍스트보다 먼저 사용자의 시선을 사로잡습니다.
- **타겟 페르소나**: VIP 골퍼

### Data-Backed Trust (데이터 기반의 신뢰)
- **정의**: 숫자(통계, 핸디캡, 리뷰)는 신뢰를 구축하는 핵심 요소입니다. 이 데이터는 복잡하지 않고 명확하게 시각화되어, 사용자가 프로의 역량을 한눈에 파악하고 신뢰할 수 있도록 돕습니다.
- **타겟 페르소나**: VIP 골퍼

## 2. Apple Human Interface Guidelines (HIG) 에서 차용하는 핵심 원칙

TEE:UP은 Apple HIG의 다음 원칙들을 웹 환경에 맞춰 차용하여 사용자에게 익숙하면서도 직관적인 경험을 제공합니다.

### 2.1. 레이아웃 (Layout)
- **큰 여백과 시각적 계층**: 요소들 간의 충분한 여백을 확보하여 시각적 혼란을 줄이고, 중요한 정보에 시선이 집중되도록 유도합니다.
- **한 화면에 하나의 핵심 액션**: 복잡성을 피하고, 사용자가 현재 화면에서 수행해야 할 가장 중요한 액션 하나에 집중할 수 있도록 디자인합니다. 이는 인지 부하를 최소화하고 의사결정을 돕습니다.

### 2.2. 타이포그래피 (Typography)
- **명확한 정보 위계**: Title, Headline, Body, Caption 등 타이포그래피의 크기, 굵기, 색상 등을 통해 정보의 중요도와 계층을 명확하게 구분합니다. 이는 콘텐츠의 가독성을 높이고 정보 탐색을 용이하게 합니다.
- **간결하고 절제된 폰트 사용**: 'Pretendard' (한국어)와 'Inter' (영어)를 주력으로 사용하며, 'JetBrains Mono'는 데이터/코드 등 특정 목적에 한해 보조적으로 사용합니다.

### 2.3. 컬러 (Color)
- **중립(Neutral) 베이스 + 1~2개 액센트 컬러**: 배경, 텍스트 등 대부분의 UI 요소에는 차분하고 고급스러운 중립 색상(calm-white, calm-cloud, calm-charcoal 등)을 사용합니다. 인터랙션 요소나 핵심 정보 강조를 위해 제한된 수의 액센트 컬러(calm-accent blue)만을 활용합니다.
- **시각적 평온함 유지**: 과도한 색상 사용을 지양하고, 전체적으로 시각적 평온함을 유지하여 사용자가 콘텐츠에 집중할 수 있도록 돕습니다.

### 2.4. Depth (깊이감)
- **그림자 최소화 및Elevation 대신 대비·타이포로 계층 표현**: 물리적인 깊이감(그림자, 입체 효과)을 최소화합니다. 대신 색상 대비, 여백, 타이포그래피의 위계 등을 통해 UI 요소 간의 계층 구조와 중요도를 표현합니다. 이는 깨끗하고 현대적인 디자인을 유지하는 데 기여합니다.

## 3. TEE:UP 전용 디자인 원칙

TEE:UP의 독특한 가치와 경험을 전달하기 위한 고유한 디자인 원칙입니다.

### 3.1. 프로 프로필은 "잡지 표지"처럼 보여줄 것
- **고품질 이미지/영상 전면 배치**: 프로의 메인 프로필 사진/영상은 페이지의 가장 눈에 띄는 곳에 배치하여, 사용자가 마치 프리미엄 골프 매거진의 표지를 넘기듯 프로필을 탐색하도록 유도합니다.
- **최소한의 UI**: 프로필 페이지에서는 내비게이션, 버튼 등 UI 요소를 최소화하고, 프로의 모습과 정보가 전면에 부각되도록 합니다.

### 3.2. 숫자(통계, 핸디캡, 리뷰)는 카드형으로 조용하게 강조
- **정제된 데이터 시각화**: 프로의 성과, 경력, 고객 평점 등 숫자로 표현되는 정보는 깔끔한 카드(Card) 형태로 제공됩니다.
- **시각적 소음 제거**: 불필요한 그래프나 애니메이션 대신, 명확한 숫자와 간결한 설명으로 정보를 전달하여 사용자가 빠르게 핵심 데이터를 파악할 수 있도록 합니다.

### 3.3. UI 요소는 최대한 사라지고, 사진·텍스트가 전면에 오게 할 것
- **몰입형 콘텐츠 경험**: 인터랙션이 필요하지 않은 상황에서는 UI 요소를 숨기거나 최소화하여, 사용자가 프로의 이미지, 영상, 설명 등 콘텐츠 자체에 완전히 몰입할 수 있도록 합니다.
- **'Show, Don't Tell' 철학**: 장황한 설명이나 복잡한 기능보다는, 잘 정돈된 시각적 스토리텔링을 통해 정보를 전달합니다.

## 4. 금지 패턴 (Anti-Patterns)

TEE:UP 디자인에서는 다음과 같은 패턴을 지양합니다.

- **과도한 그라디언트 / 네온 컬러**: 시각적 피로도를 높이고 고급스러운 이미지를 해칠 수 있는 화려하고 인위적인 색상 사용을 금지합니다.
- **카드 안에 카드, 그림자 중첩, 복잡한 보더**: 디자인의 복잡성을 증가시키고 'Calm Luxury' 원칙에 위배되는 중첩된 레이아웃, 과도한 그림자, 복잡한 테두리 사용을 지양합니다. 간결하고 명확한 구성을 유지합니다.

## 5. 예시 (웹 환경: Desktop & Mobile Web)

### 5.1. 프로필 페이지 (Desktop)
- **레이아웃**: 넓은 여백, 중앙 집중형 콘텐츠, 좌우에 고품질 이미지 배치.
- **타이포그래피**: 프로 이름은 크고 굵게, 주요 정보는 헤드라인으로 명확히. 상세 설명은 Body 텍스트로 가독성 확보.
- **데이터 카드**: 프로필 하단에 "레슨 완료 횟수", "평균 핸디캡 감소", "리뷰 평점" 등을 3~4개의 간결한 카드 형태로 배치.
- **CTA**: "문의하기" 버튼은 페이지 하단에 Sticky 형태로 존재하며, 최소한의 디자인으로 콘텐츠를 가리지 않음.

### 5.2. 프로필 페이지 (Mobile Web)
- **레이아웃**: 스크롤 기반의 세로형 콘텐츠 배치. 중요 이미지/영상이 화면 상단을 크게 차지.
- **타이포그래피**: 데스크톱과 동일한 정보 위계 유지. 작은 화면에 맞춰 폰트 크기 조정.
- **데이터 카드**: 데스크톱과 유사하게 카드 형태로 제공되나, 모바일 화면에 맞춰 1~2열로 유동적으로 배치.
- **CTA**: "문의하기" 버튼은 하단 탭 바 또는 Sticky 버튼으로 유지.

### 5.3. 목록 페이지 (Desktop & Mobile Web)
- **레이아웃**: 그리드 형태의 프로 카드 목록. 각 카드 간 충분한 여백.
- **프로 카드**: 프로의 얼굴/스윙 이미지, 이름, 핵심 태그, 평점 등을 간결하게 표시. 그림자 최소화.
- **필터/검색**: 상단에 미니멀한 검색 바와 필터 옵션. 불필요한 아이콘이나 장식 배제.
